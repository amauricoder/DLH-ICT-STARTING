In this module we will use HTML and CSS.

      A  certain HTML element  allows
              text to be displayed as it is entered in the editor.
                    This is called
                            'preformatted'.

Example of a CSS rule:

       body {
              color: #ff0000;
              font-family: Arial, Verdana; }

Very important words or text passages are highlighted.

In mathematics, some text is superscripted or subscripted, just like π2.

The text in the footer is often printed small.

Words or text passages that are emphasized when speaking can also be emphasized in HTML.

A text can be highlighted, like with a text marker.

Of course you can combine all these HTML elements. Whether this always makes sense,
however, is questionable. In any case, make sure that the elements are always nested
correctly and always follow the basic rule: the first opened element is closed last.
