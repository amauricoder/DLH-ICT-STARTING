aumentando linhas com colspan

Para mudar tamanho das linhas utilizamos colspan
por exemplo, para aumentar o tamanho da linha em 2
devemos colocar = "2" na linha desejada
*ver no codigo HTML o exemplo*

Aumentando colunas com rowspan

O principio é o mesmo que o colspan.
colocar rowspan = "numero" dentro da coluna
que voce deseja aumentar. Ex. >td rowspan ="2"> conteudo 2 - linha 2
aumentara a coluna da linha 2 em 2.
*ver codigo html para exemplo*.
